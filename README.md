# 💰 Cash Flow – Salary & Expense Tracker

A modern and responsive **Salary & Expense Tracker** built using **HTML5, CSS3, and Vanilla JavaScript**. This application helps users manage their monthly salary, track expenses, visualize spending patterns, and monitor their remaining balance with an intuitive dashboard.

---

## 🚀 Live Demo

(https://bejewelled-selkie-86e39e.netlify.app/)

## 📌 Features

### ✅ Phase 1 – Core Features

- Add Monthly Salary
- Add Expenses
- Input Validation
- Real-time Expense List
- Total Salary Display
- Total Expense Calculation
- Remaining Balance Calculation
- Responsive User Interface

---

### ✅ Phase 2 – Advanced Features

- Data Persistence using Local Storage
- Delete Expense Functionality
- Automatic Balance Recalculation
- Interactive Pie Chart using Chart.js
- Clean Card-based Dashboard
- Responsive Table Layout

### ✅ Phase 3 – Additional Features

- PDF Report Generation using jsPDF
- Low Balance Warning
- Currency Selection Support
